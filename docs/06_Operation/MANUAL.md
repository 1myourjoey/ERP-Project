## 1. 시스템 개요

### 소개

VC ERP는 벤처캐피탈 백오피스 1인 관리자를 위한 업무 관리 시스템입니다.
투자심의위원회부터 사후관리까지, 조합 결성부터 월보고까지 모든 VC 업무를 체계적으로 관리합니다.

### 기술 스택

- **백엔드**: FastAPI + SQLAlchemy + SQLite
- **프론트엔드**: React + Vite + TailwindCSS
- **API 문서**: http://localhost:8000/docs (Swagger UI)

### 접속 방법

1. 백엔드 서버 시작:
   ```
   cd backend
   python -m uvicorn main:app --port 8000
   ```

2. 프론트엔드 서버 시작:
   ```
   cd frontend
   npm run dev
   ```

3. 브라우저에서 접속: **http://localhost:5173**

### 운영 조합

| 조합 | 유형 |
|------|------|
| PEX투자조합 | 벤처투자조합 |
| 하이테크 1호 | 벤처투자조합 |
| 파마테크 2호 | 벤처투자조합 |
| 메디테크 3호 | 벤처투자조합 |
| 고유계정 | 고유계정 |
| 농모태 | 농금원 |

## 3. 대시보드

### 화면 구성

대시보드는 오늘 날짜와 요일을 표시하며, 4개 섹션으로 나뉩니다:

- **오늘**: 오늘 마감인 작업 (총 예상시간 표시)
- **내일**: 내일 마감인 작업
- **이번 주**: 이번 주 내 마감 작업
- **다가오는 일정**: 향후 2개월 내 일정

### 진행 중인 워크플로우

상단에 현재 활성화된 워크플로우가 표시됩니다:
- 워크플로우 이름, 진행률 (예: 3/7), 다음 단계

### 클릭 네비게이션

- **작업 카드 클릭** → 작업 보드 페이지로 이동
- **워크플로우 카드 클릭** → 워크플로우 페이지로 이동 (해당 인스턴스 자동 펼침)

## 4. 작업 보드 (아이젠하워 매트릭스)

### 4분면 분류

| 분면 | 기준 | 색상 | 예시 |
|------|------|------|------|
| **Q1** | 긴급 & 중요 | 빨강 | 투심위 서류 마감, 계약 날인 |
| **Q2** | 중요 & 비긴급 | 파랑 | 내부보고회 준비, 시스템 개선 |
| **Q3** | 긴급 & 비중요 | 주황 | 단순 회의, 위임 가능 업무 |
| **Q4** | 비긴급 & 비중요 | 회색 | 나중에 처리, 제거 가능 |

### 작업 추가

1. 해당 분면 하단의 **"+ 추가"** 버튼 클릭
2. 작업명 입력 (필수)
3. 마감일 선택 (날짜)
4. 예상 소요시간 입력 (예: `2h`, `30m`, `1h30m`)
5. **추가** 버튼 클릭 또는 Enter

### 작업 편집

1. 작업 항목에 마우스를 올려 **연필 아이콘** 클릭
2. 편집 모달에서 수정:
   - 작업명, 마감일, 예상시간
   - 분면 변경 (Q1~Q4 드롭다운)
   - 메모, 위임 대상
3. **저장** 클릭

### 작업 완료

1. 작업 항목 좌측의 **원형 체크박스** 클릭
2. 실제 소요시간 입력 (예: `1h30m`)
3. **완료** 버튼 클릭
4. 작업이 완료 상태로 변경되며, 대시보드에서 제거됩니다

### 작업 삭제

1. 작업 항목에 마우스를 올려 **휴지통 아이콘** 클릭
2. 확인 대화상자에서 승인

### 상태 필터

상단 우측에서 필터를 전환할 수 있습니다:
- **진행**: 진행 중인 작업만 표시 (기본값)
- **전체**: 모든 작업 표시
- **완료**: 완료된 작업만 표시

## 5. 워크플로우

### 5.1 워크플로우 템플릿

워크플로우 템플릿은 반복되는 업무 프로세스의 표준 절차입니다.

#### 기본 제공 템플릿

| 템플릿 | 카테고리 | 단계 | 소요기간 |
|--------|----------|------|----------|
| 투자심의위원회 | 투자 | 7단계 | 1~2주 |
| 투자계약 체결 | 투자 | 8단계 | 3~5일 |
| 투자 후 서류처리 | 투자 | 6단계 | 2~3주 |
| 조합 결성 - 고유번호증 발급 | 조합결성 | 4단계 | 3~5일 |
| 조합 결성 - 수탁계약 체결 | 조합결성 | 4단계 | 3~5일 |
| 조합 결성 - 결성총회 개최 | 조합결성 | 6단계 | 2~3주 |
| 내부보고회 | 정기업무 | 4단계 | 2주 |
| 월보고 | 정기업무 | 2단계 | 5일 |

#### 템플릿 상세 보기

1. 좌측 "템플릿" 목록에서 항목 클릭
2. 상세 정보 표시:
   - **단계**: 순서, 이름, 타이밍 (D-7, D-day, D+2 등), 예상시간
   - **필요 서류**: 각 단계에 필요한 서류 목록
   - **주의사항**: 업무 수행 시 유의점

#### 새 템플릿 만들기

1. **"+ 새 템플릿"** 버튼 클릭
2. 기본 정보 입력: 이름, 트리거 설명, 카테고리, 총 소요기간
3. 단계 추가:
   - 순서, 단계명, 타이밍 (D-day 기준 오프셋), 예상시간
   - 분면 설정 (기본값 Q1)
   - 메모 (선택)
4. 필요 서류 추가: 서류명, 필수 여부, 타이밍
5. 주의사항 추가
6. **저장** 클릭

#### 템플릿 편집/삭제

- **편집**: 상세 보기에서 **편집** 버튼 클릭 → 수정 후 저장
- **삭제**: **삭제** 버튼 클릭 → 확인 (활성 인스턴스가 있으면 삭제 불가)

### 5.2 워크플로우 실행

#### 워크플로우 시작

1. 템플릿 상세에서 **"워크플로우 시작"** 클릭
2. 정보 입력:
   - **인스턴스 이름**: 예) "A기업 투자계약", "하이테크1호 2월 월보고"
   - **기준일**: D-day에 해당하는 날짜 (예: 투심위 개최일)
   - **메모**: 선택사항
3. **시작** 클릭

시작하면:
- 각 단계의 날짜가 기준일 기준으로 자동 계산됩니다 (주말 자동 건너뜀)
- 각 단계가 작업 보드에 자동으로 Task로 등록됩니다
- 작업명: `[인스턴스명] 단계명` 형식

#### 단계 완료 처리

1. 우측 "진행 중" 목록에서 워크플로우 클릭하여 펼침
2. 완료할 단계의 **체크박스** 클릭
3. 연결된 Task도 자동으로 완료 처리됩니다
4. 모든 단계 완료 시 워크플로우 인스턴스도 자동 완료됩니다

#### 워크플로우 취소

- 펼친 인스턴스 하단의 **"워크플로우 취소"** 클릭
- 미완료 Task도 함께 삭제됩니다

## 6. 업무일지

### 업무 기록 추가

1. 우측 상단 **"+ 기록 추가"** 클릭
2. 입력:
   - **날짜**: 업무 수행일
   - **카테고리**: 드롭다운에서 선택
   - **제목**: 업무 제목
   - **내용**: 상세 설명 (선택)
   - **예상 시간 / 실제 시간 / 차이**: 시간 추적
3. **저장** 클릭

### 카테고리

| 카테고리 | 설명 |
|----------|------|
| 투심위 | 투자심의위원회 관련 |
| 투자계약 | 투자계약 체결 관련 |
| 투자계약 후속 | 운용지시 등 후속 처리 |
| 투자 후 등록 | 바이블, VICS 등록 등 |
| 조합감사 | 기말감사 관련 |
| 내부보고회 | 분기 내부보고회 |
| 연말정산 | 연말정산 관련 |
| 총무 | 총무 업무 |
| 바이블 목록 제작 | 피투자기업 서류 정리 |
| LP바인더 | LP 관련 서류 정리 |
| 체크리스트 | 체크리스트 작업 |
| 시스템 설정 | ERP/시스템 관련 |
| 정기 업무 | 월보고 등 정기 업무 |

### 업무 기록 편집

1. 기록 항목 클릭하여 펼침
2. **연필 아이콘** 클릭
3. 내용 수정 + 상세/교훈/후속업무 추가/수정
4. **저장** 클릭

### 날짜별 그룹

업무일지는 날짜별로 그룹화되어 표시됩니다. 최신 날짜가 상단에 위치합니다.

## 7. 펀드 관리

### 펀드 목록

모든 운영 조합을 카드 형식으로 표시합니다:
- 펀드명, 유형, 상태
- 약정총액, 출자이행률
- 수탁사, 고유번호

### 펀드 상세

펀드 카드 클릭 시 상세 페이지:

#### LP 관리
- LP 목록 (이름, 유형, 약정금액, 납입금액, 연락처)
- LP 추가/편집/삭제
- LP 유형: 개인, 법인, 집합투자기구, 창업기획자, LLC 등

#### 투자 현황
- 해당 펀드의 투자건 목록
- 투자 총액, 투자 기업 수

### 펀드 추가

1. **"+ 펀드 추가"** 클릭
2. 정보 입력: 펀드명, 유형, 결성일, 약정총액, 수탁사 등
3. **저장** 클릭

## 8. 투자 관리

### 투자건 목록

모든 투자건을 테이블/카드 형식으로 표시합니다:
- 펀드별, 기업별, 상태별 필터
- 투자유형, 투자금액, 지분율, 계약일

### 투자건 등록

1. **"+ 투자건 등록"** 클릭
2. 정보 입력:
   - **펀드 선택**: 드롭다운
   - **기업 선택**: 기존 기업 선택 또는 새 기업 등록
   - **투자 유형**: 보통주, CB, RCPS 등
   - **투자금액, 주식수, 주당단가, Pre-valuation**
   - **계약일, 납입일**
3. **저장** 클릭

### 포트폴리오 기업

포트폴리오 기업 목록에서:
- 기업 정보 (사업자번호, 대표자, 업종)
- VICS 등록 여부
- 해당 기업의 전체 투자 내역 (펀드 간 교차 투자 포함)

### 투자 서류 관리

투자건 상세 페이지에서 서류 관리:
- 서류명, 서류유형, 상태 (대기/수집/검증/수탁사발송)
- 수집일, 비고
- 상태 변경으로 서류 진행 추적

## 9. 업무 프로세스 가이드

### 9.1 신규 투자 처리 (End-to-End)

전체 흐름: **투심위 → 계약 → 사후 서류**

#### Step 1: 투심위 워크플로우 시작
1. 워크플로우 > 투자심의위원회 템플릿 > **워크플로우 시작**
2. 인스턴스명: "[기업명] 투심위"
3. 기준일: 투심위 개최 예정일

자동 생성되는 Task:
- D-7: 추가출자금 납입 통보 (납입 요청 공문 + 조합 통장사본)
- D-3: 투심 전 ERP 등록 (투심 보고서 + 투자계약서 + 준법감시보고서)
- D-day: 투심위 개최 / 서류 사인(13:00) / ERP 등록 / 농금원 결과보고서
- D+2: 투심위 회의록 작성

#### Step 2: 투자계약 워크플로우 시작
1. 투심위 통과 후, 워크플로우 > 투자계약 체결 > **워크플로우 시작**
2. 인스턴스명: "[기업명] 투자계약"
3. 기준일: 계약 예정일

#### Step 3: 투자 후 서류처리 워크플로우 시작
1. 투자금 출금 후, 워크플로우 > 투자 후 서류처리 > **워크플로우 시작**
2. 기준일: 투자금 출금일

### 9.2 조합 결성 프로세스

3개 워크플로우를 순차적으로 실행:

1. **고유번호증 발급** (기준일: 결성 시작일)
2. **수탁계약 체결** (기준일: 고유번호증 수령 후)
3. **결성총회 개최** (기준일: 수탁계약 체결일)

### 9.3 월보고 프로세스

매월 초:
1. 워크플로우 > 월보고 > **워크플로우 시작**
2. 인스턴스명: "2월 월보고" (해당 월)
3. 기준일: 매월 5일 (내부 마감)

| 보고처 | 내부 마감 | 공식 마감 |
|--------|-----------|-----------|
| 농금원 | 매월 5일 | 매월 7일 |
