## 1. 시스템 개요

### 소개

VC ERP는 벤처캐피탈 백오피스 1인 관리자를 위한 통합 업무 관리 시스템입니다.
투자심의위원회부터 사후관리까지, 조합 결성부터 월보고까지 모든 VC 업무를 체계적으로 관리합니다.

### 기술 스택

| 구성 요소 | 기술 |
|-----------|------|
| 백엔드 | FastAPI + SQLAlchemy + SQLite |
| 프론트엔드 | React 18 + Vite + TailwindCSS |
| 상태 관리 | TanStack Query (React Query) |
| 라우팅 | React Router v6 |
| API 문서 | http://localhost:8000/docs (Swagger UI) |

### 접속 방법

1. 백엔드 서버 시작:
   ```bash
   cd backend
   python -m uvicorn main:app --port 8000
   ```

2. 프론트엔드 개발 서버 시작:
   ```bash
   cd frontend
   npm run dev
   ```

3. 브라우저에서 접속: **http://localhost:5173**

## 3. 대시보드

### 화면 구성

대시보드는 크게 **좌측 업무 영역**과 **우측 현황 탭**으로 나뉩니다.

#### 좌측: 업무 보드 요약

| 섹션 | 내용 |
|------|------|
| 오늘 | 오늘 마감인 작업 (총 예상시간 표시) |
| 내일 | 내일 마감인 작업 |
| 이번 주 | 이번 주 마감 작업 (오늘/내일 제외) |
| 다가오는 일정 | 향후 2개월 내 일정 |
| 완료 | 오늘/이번주/전주 완료된 작업 (필터 전환 가능) |
| 진행 중 워크플로우 | 활성 워크플로우 진행률 + 현재 단계 표시 |

#### 우측: 현황 탭 (4개)

| 탭 | 내용 |
|----|------|
| **조합** | 전체 조합 현황 카드 (약정총액, 투자건수 등) |
| **통지** | 다가오는 LP 통지 일정 (D-day 표시) |
| **보고** | 정기 보고 마감 일정 + 업무연동 버튼 |
| **서류** | 미수집 서류 현황 |

### 주요 기능

- **QuickAdd**: "오늘" 섹션 우측 "+" 버튼으로 빠른 업무 추가 (제목, 마감일, 예상시간, 조합, 카테고리)
- **리스트 팝업 모달**: 각 섹션 접기/펼치기 또는 "더보기" 클릭 시 전체 리스트 모달 표시
- **업무보드 확인**: 대시보드에서 업무 클릭 시 업무 보드로 이동 후 해당 업무 깜빡임 + 자동 스크롤
- **완료 취소**: 완료된 업무에 "되돌리기" 버튼으로 완료 취소 가능
- **6개 상단 카드**: 오늘/내일/이번주/워크플로/서류/보고 건수 숫자 표시 → 클릭 시 리스트 팝업

## 4. 업무 보드 (아이젠하워 매트릭스)

### 4분면 분류

| 분면 | 기준 | 색상 | 예시 |
|------|------|------|------|
| **Q1** | 긴급 & 중요 | 빨강 | 투심위 서류 마감, 계약 날인 |
| **Q2** | 중요 & 비긴급 | 파랑 | 내부보고회 준비, 시스템 개선 |
| **Q3** | 긴급 & 비중요 | 주황 | 단순 회의, 위임 가능 업무 |
| **Q4** | 비긴급 & 비중요 | 회색 | 나중에 처리, 제거 가능 |

### 작업 추가

1. 해당 분면 하단의 **"+ 추가"** 버튼 클릭
2. 작업명 입력 (필수)
3. 마감일 선택 (미니 캘린더에서 날짜 선택)
4. 마감 시간 선택 (시간 드롭다운)
5. 예상 소요시간 입력 (예: `2h`, `30m`, `1h30m`)
6. **카테고리** 선택: 투자실행, LP보고, 사후관리, 규약/총회, 서류관리, 일반
7. **조합 선택** (드롭다운): 특정 조합에 연결
8. **워크플로우 템플릿** 선택 시: 템플릿의 전체 단계가 자동 생성됨
9. **추가** 버튼 클릭 또는 Enter

### 작업 편집

1. 작업 항목 우측의 **"수정"** 버튼 클릭 (파란색)
2. 편집 모달에서 수정:
   - 작업명, 마감일, 마감시간, 예상시간
   - 분면 변경 (Q1~Q4 드롭다운)
   - 메모, 위임 대상
3. **저장** 클릭

### 작업 완료

1. 작업 항목 우측의 **"완료"** 버튼 클릭 (초록색)
2. CompleteModal 팝업에서 **실제 소요시간** 입력
3. **완료** 버튼 클릭
4. 작업이 완료 상태로 변경됨

### 작업 삭제

- 작업 항목에 마우스를 올려 **휴지통 아이콘** 클릭 (hover 시 표시)

### 상태 필터

상단 우측에서 필터 전환:
- **진행**: 진행 중인 작업만 표시 (기본)
- **전체**: 모든 작업 표시
- **완료**: 완료된 작업만 표시

### 뷰 전환

- **기본 보기 / 캘린더 보기**: 상단 토글로 전환
- 캘린더 보기: 월간 달력에 작업을 날짜별로 표시

### 워크플로우 그룹

- 워크플로우로 생성된 작업은 접이식 그룹 카드로 표시
- 진행률 뱃지 (예: `3/7`), 현재 단계 표시
- 그룹 클릭으로 펼침/접기

### 드래그 앤 드롭

- 작업을 다른 분면으로 드래그하여 이동 가능

## 5. 업무 기록 (업무일지)

### 업무 기록 추가

1. 우측 상단 **"+ 기록 추가"** 클릭
2. 입력:
   - **날짜**: 업무 수행일
   - **카테고리**: 드롭다운 선택 (투심위, 투자계약, 투자계약 후속, 투자 후 등록, 조합감사, 내부보고회, 연말정산, 총무, 바이블 목록 제작, LP바인더, 체크리스트, 시스템 설정, 정기 업무)
   - **제목**: 업무 제목
   - **내용**: 상세 설명 (선택)
   - **예상 시간 / 실제 시간**: 시간 추적
3. **저장** 클릭

### 업무 기록 편집

1. 기록 항목 클릭하여 펼침
2. **연필 아이콘** 클릭
3. 내용 수정 + 상세/교훈/후속업무 추가/수정
4. **저장** 클릭

### 날짜별 그룹

업무일지는 날짜별로 그룹화되어 표시됩니다. 최신 날짜가 상단에 위치합니다.

## 6. 체크리스트

### 개요

투자건별 서류 체크리스트를 생성하고 관리합니다. 투자 후 서류 수집 진행률을 시각적으로 추적할 수 있습니다.

### 체크리스트 생성

1. **"+ 체크리스트 추가"** 클릭
2. 입력:
   - **이름**: 체크리스트 제목 (예: "A기업 투자 서류")
   - **카테고리**: 분류
   - **투자건 연결**: 드롭다운에서 투자건 선택 (선택)
3. **저장** 클릭

### 항목 관리

- 체크리스트 선택 후 상세 패널에서 **항목 추가**
- 각 항목: 순번, 이름, 필수 여부, 체크 상태, 비고
- 체크 토글로 수집 완료 표시
- 항목별 수정/삭제 가능

## 7. 조합 개요

### 개요

전체 운영 조합의 요약 현황을 한 화면에서 확인합니다.

### 화면 구성

- **조합별 카드**: 각 조합의 약정총액, 투자건수, 기업수 표시
- **금액 표기**: 전체 숫자 + 한글 단축 표기 동시 표시 (예: `5,000,000,000원` / `50.0억원`)

### 엑셀 다운로드

- 우측 상단 **엑셀 다운로드** 버튼으로 전체 조합 현황을 `.xlsx` 파일로 내보내기
- 포함 항목: 조합명, 유형, 상태, 약정총액, 투자건수, 기업수

## 8. 조합 관리

### 조합 목록

모든 운영 조합을 **카드 형식**으로 표시합니다:
- 조합명, 유형 (드롭다운 선택), 상태 뱃지
- **자동 날짜 표시**: 상태에 따라 다르게 표시
  - `결성예정` (forming) → 결성예정: 결성일
  - `운용 중` (active) → 운용 중: 현재일 (N일째)
  - `해산/청산 완료` (dissolved/liquidated) → 해산/청산 완료: 만기일 또는 해산일
- **약정총액**: 전체 숫자 + 한글 표기 (예: `5,000,000,000원` / `50.0억원`)
- **투자건수**: 해당 조합의 투자 건수

### 조합 추가

1. **"+ 조합 추가"** 클릭
2. 정보 입력:
   - **조합명** (필수)
   - **조합 유형**: 드롭다운 선택 (투자조합, 벤처투자조합, 신기술투자조합, 사모투자합자회사(PEF), 창업투자조합, 기타)
   - **결성일**: 날짜 선택
   - **상태**: 드롭다운 선택 (결성예정, 운용 중, 해산, 청산 완료)
   - **GP, Co-GP, 신탁사**
   - **총 약정액**: 숫자 입력
3. **저장** 클릭

### 조합 카드 클릭

카드 클릭 시 **조합 상세 페이지** (`/funds/:id`)로 이동합니다.

## 9. 조합 상세 (FundDetail)

### 접근

조합 관리 → 조합 카드 클릭 → 조합 상세 페이지

### 기본 정보 섹션

- 조합명, 유형, 결성일, 상태, GP, Co-GP, 신탁사
- 약정총액, 관리보수율, 성과보수율 등 핵심 조건 표시
- **수정 모드**: 연필 아이콘 클릭 → 인라인 수정 가능

### LP 관리

| 필드 | 설명 |
|------|------|
| LP명 | 출자자 이름 |
| 유형 | 개인, 법인, 집합투자기구, 창업기획자, LLC 등 |
| 약정금액 | 원 단위 숫자 입력 |
| 납입금액 | 실제 납입 금액 |
| 연락처 | 이메일 또는 전화 |

- LP 추가/편집/삭제 가능

### 투자 현황

- 해당 조합의 투자건 목록 (카드 뷰 / 테이블 뷰 토글)
- 투자 총액, 투자 기업 수

### 출자 이력

- 출자 요청 이력 테이블
- 출자 방식 드롭다운: 정기출자, 추가출자, 매니저클로징, 기타

### 주요 조건 (Key Terms)

- 관리보수, 성과보수, 존속기간, 투자기간 등 핵심 조건 관리

## 12. 워크플로우

### 12.1 워크플로우 템플릿

워크플로우 템플릿은 반복되는 업무 프로세스의 표준 절차입니다.

#### 기본 제공 템플릿

| 템플릿 | 카테고리 | 단계 | 소요기간 |
|--------|----------|------|----------|
| 투자심의위원회 | 투자 | 7단계 | 1~2주 |
| 투자계약 체결 | 투자 | 8단계 | 3~5일 |
| 투자 후 서류처리 | 투자 | 6단계 | 2~3주 |
| 조합 결성 - 고유번호증 발급 | 조합결성 | 4단계 | 3~5일 |
| 조합 결성 - 수탁계약 체결 | 조합결성 | 4단계 | 3~5일 |
| 조합 결성 - 결성총회 개최 | 조합결성 | 6단계 | 2~3주 |
| 내부보고회 | 정기업무 | 4단계 | 2주 |
| 월보고 | 정기업무 | 2단계 | 5일 |

#### 템플릿 목록

- 좌측 패널에 템플릿 카드 목록
- 각 카드에 **원형 체크박스** (우측 상단): 클릭 시 해당 템플릿 선택/해제 + 상세 패널 자동 열림/닫힘 (단일 선택)
- 카드 클릭으로도 선택 가능

#### 템플릿 상세 보기

템플릿 선택 시 우측에 상세 정보 표시:
- **단계**: 순서, 이름, 타이밍 (D-7, D-day, D+2 등), 예상시간
- **필요 서류**: 각 단계에 필요한 서류 목록
- **주의사항**: 업무 수행 시 유의점

#### 새 템플릿 만들기

1. **"+ 새 템플릿"** 버튼 클릭
2. 기본 정보 입력: 이름, 트리거 설명, 카테고리, 총 소요기간
3. 단계 추가: 순서, 단계명, 타이밍, 예상시간, 분면, 메모
4. 필요 서류 추가: 서류명, 필수 여부, 타이밍
5. 주의사항 추가
6. **저장** 클릭

#### 템플릿 편집/삭제

- **편집**: 상세 보기에서 **편집** 버튼 클릭 → 수정 후 저장
- **삭제**: **삭제** 버튼 클릭 → 확인 (활성 인스턴스가 있으면 삭제 불가)

### 12.2 워크플로우 실행

#### 워크플로우 시작

1. 템플릿 상세에서 **"워크플로우 시작"** 클릭
2. 정보 입력:
   - **인스턴스 이름**: 예) "A기업 투자계약", "하이테크1호 2월 월보고"
   - **기준일**: D-day에 해당하는 날짜 (예: 투심위 개최일)
   - **조합 선택**: (선택) 연결할 조합
   - **기업 선택**: (선택) 연결할 기업
   - **메모**: 선택사항
3. **시작** 클릭

시작하면:
- 각 단계의 날짜가 기준일 기준으로 자동 계산됩니다 (주말 자동 건너뜀)
- 각 단계가 업무 보드에 Task로 자동 등록됩니다
- 작업명: `[인스턴스명] 단계명` 형식

#### 단계 완료 처리

1. 우측 "진행 중" 목록에서 워크플로우 클릭하여 펼침
2. 완료할 단계의 **원형 체크버튼** 클릭
3. 완료 시간이 자동 기록되어 표시됩니다 (예: "2/15 14:30")
4. **"되돌리기"** 링크로 완료 취소 가능
5. 연결된 Task도 자동으로 완료 처리됩니다
6. 모든 단계 완료 시 워크플로우 인스턴스도 자동 완료

#### 워크플로우 취소

- 펼친 인스턴스 하단의 **"워크플로우 취소"** 클릭
- 미완료 Task도 함께 삭제됩니다

#### 진행 중 인스턴스 수정

- **"수정"** 버튼 클릭 → 인스턴스 이름, 트리거 날짜, 메모 인라인 수정

#### 인쇄 기능

- **"인쇄"** 버튼 → 한글(Malgun Gothic) 폰트로 깔끔한 인쇄 출력

## 13. 회수 관리 (Exits)

### 개요

투자 회수(Exit) 과정을 관리합니다. 회수위원회 결의와 매각거래 두 영역으로 구성됩니다.

### 회수위원회

- **위원회 등록**: 회의일, 상태(예정/심의/승인/반려), 회수전략, 메모
- **대상 펀드/투자건 연결**: 위원회에 대상 조합과 투자건을 연결

### 매각거래 (Exit Trades)

| 필드 | 설명 |
|------|------|
| 위원회 연결 | 승인된 위원회와 연결 (선택) |
| 거래일 | 매각 날짜 |
| 유형 | IPO 매도, M&A, 인수합병, 전환, 청산배분, 배당 |
| 금액 | 매각 금액 |
| 주식수 | 매각 주식 수 |
| 수수료 | 거래 수수료 |
| 실현손익 | 실현 손익 금액 |

## 19. 조합 운영 (FundOperations)

### 개요

출자, 배분, 총회 운영과 성과지표를 관리합니다.

### 조합 선택

- 페이지 상단의 **눈에 띄는 카드형 셀렉터**로 조합 선택
- 아이콘 + "조합 선택" 라벨 + 큰 드롭다운
- **"조합 상세 보기"** 버튼으로 조합 관리 페이지 연결

### 성과지표 카드

| 지표 | 설명 |
|------|------|
| 납입 총액 | 실제 납입된 금액 합계 |
| 투자 총액 | 투자 실행 금액 합계 |
| 배분 총액 | LP 배분 금액 합계 |
| 잔여 가치 | 미회수 투자 잔여 가치 |
| TVPI | 총 가치 대비 납입 배수 |
| DPI | 배분 대비 납입 배수 |
| RVPI | 잔여 가치 대비 납입 배수 |
| IRR | 내부수익률 |

기준일 선택으로 특정 시점의 성과지표 확인 가능

### 출자 관리

| 항목 | 설명 |
|------|------|
| 출자 등록 | 날짜, 출자방식(정기/추가/매니저클로징/기타), 총액 |
| LP 내역 | 출자별 LP별 납입 내역 (미납/납입 상태) |
| 수정/삭제 | 각 출자 건 및 LP 항목 개별 수정/삭제 |

### 배분 관리

| 항목 | 설명 |
|------|------|
| 배분 등록 | 날짜, 유형(현금/원금/수익/잔여자산), 원금총액, 수익총액 |
| LP 배분 내역 | 배분별 LP별 원금/수익 배분 항목 |
| 수정/삭제 | 각 배분 건 및 LP 항목 개별 수정/삭제 |

### 총회 관리

| 항목 | 설명 |
|------|------|
| 총회 등록 | 날짜, 유형(결성/정기/임시), 상태, 안건 |

| 수정/삭제 | 각 총회 건 개별 수정/삭제 |

## 20. 서류 현황 (Documents)

### 개요

전체 투자건의 서류 수집 상태를 한눈에 추적합니다.

### 요약 카드

- **전체 서류 수**: 등록된 모든 서류 건수
- **미수집 서류 수**: 아직 수집되지 않은 서류
- **수집률**: 수집 완료 비율 (%)

### 필터

- **상태**: 전체 / 미수집 / 요청중 / 수집완료
- **조합**: 전체 조합 중 선택
- **회사**: 전체 회사 중 선택

### 서류 상태 변경

테이블 내 드롭다운으로 상태 직접 변경:
미수집 → 요청중 → 검토중 → 수집완료

### D-day 표시

서류 마감일 기준으로 D-day 뱃지 자동 표시 (지연/긴급/일반)

| 설명 | 상세 내용 |
| 상태 | pending / completed |
| 업무 연결 | Task와 연결 (선택) |

### 뷰 모드

- **월간 보기**: 달력 그리드에 일정 표시
- **주간 보기**: 주간 타임라인
- **일간 보기**: 일별 상세 일정

### 일정 등록

달력의 날짜 셀 클릭 → 일정 생성 폼 표시

## 23. 업무 프로세스 가이드

### 23.1 신규 투자 처리 (End-to-End)

전체 흐름: **투심위 → 계약 → 사후 서류**

#### Step 1: 투심위 워크플로우 시작
1. 워크플로우 > 투자심의위원회 템플릿 > **워크플로우 시작**
2. 인스턴스명: "[기업명] 투심위"
3. 기준일: 투심위 개최 예정일

자동 생성되는 Task:
- D-7: 추가출자금 납입 통보
- D-3: 투심 전 ERP 등록
- D-day: 투심위 개최 / 서류 사인 / ERP 등록 / 결과보고서
- D+2: 투심위 회의록 작성

#### Step 2: 투자계약 워크플로우 시작
1. 투심위 통과 후, 워크플로우 > 투자계약 체결 > **워크플로우 시작**
2. 인스턴스명: "[기업명] 투자계약"
3. 기준일: 계약 예정일

#### Step 3: 투자 후 서류처리 워크플로우 시작
1. 투자금 출금 후, 워크플로우 > 투자 후 서류처리 > **워크플로우 시작**
2. 기준일: 투자금 출금일

### 23.2 조합 결성 프로세스

3개 워크플로우를 순차적으로 실행:

1. **고유번호증 발급** (기준일: 결성 시작일)
2. **수탁계약 체결** (기준일: 고유번호증 수령 후)
3. **결성총회 개최** (기준일: 수탁계약 체결일)

### 23.3 월보고 프로세스

매월 초:
1. 워크플로우 > 월보고 > **워크플로우 시작**
2. 인스턴스명: "2월 월보고" (해당 월)
3. 기준일: 매월 5일 (내부 마감)

| 보고처 | 내부 마감 | 공식 마감 |
|--------|-----------|-----------|
| 농금원 | 매월 5일 | 매월 7일 |
| 벤처협회 (VICS) | 매월 5일 | 매월 9일 |

### 23.4 분기 내부보고회

1. 워크플로우 > 내부보고회 > **워크플로우 시작**
2. 기준일: 보고회 예정일
3. D-10부터 자료 취합 → D-5 초안 → D-day 진행 → D+2 회의록

### 23.5 연간 주요 일정

| 시기 | 업무 | 비고 |
|------|------|------|
| 1~2월 | 기말감사 | 회계사 현장 감사 |
| 1~2월 | 연말정산 | 세무법인 연계 |
| 3월 | 정기 총회 | 조합별 개최 |
| 3월 | 권리명세서 작성 | 홈택스 업로드 |
| 분기별 | 내부보고회 | 3Q, 4Q 등 |
| 연중 | 법정의무교육 | AML 포함 (11월) |

### 날인 서류 체크리스트

- [ ] 투심위 결과보고서
- [ ] 투심위 의사록
- [ ] 준법감시보고서
- [ ] 의무기재사항 확인서 (준법감시인)
- [ ] 조합원 명부
- [ ] 투자계약서 (간인 + 인감)

### 간인 순서

투자계약서 3부 간인 순서:
1. **조합 인감** (첫 번째)
2. **피투자사** (두 번째)
3. **이해관계인** (세 번째)

배부: 가운데 양옆 날인 → 피투자사, 맨 오른쪽 → 이해관계인

### 유효기간 확인 서류

다음 서류는 **3개월 이내 발급본**이어야 합니다:
- 법인인감증명서
- 법인등기부등본

- 주식 수가 투자계약서와 일치하는지
- 주주명부 비교: 투자 전/후 지분율이 투심위 기반과 일치하는지
