# PRD 01 — 게이트웨이 대시보드

> **상위 문서:** [PRD_MASTER.md](./PRD_MASTER.md)
> **관련 페이지:** `/dashboard`
> **상태:** 🟡 개발 중 | **최종 업데이트:** 2026-02-22

---

## 1. 기능 개요

| 항목 | 내용 |
|---|---|
| **목적** | 1~2인 VC 관리팀이 출근 직후 30초 안에 오늘의 핵심 현황을 파악하는 경량 허브 |
| **가정** | 사용자는 대시보드에서 깊은 업무 처리를 원하지 않는다. 클릭 → 해당 페이지로 즉시 이동. |
| **범위 밖** | 대시보드 내 직접 Task 생성/편집, 인라인 데이터 수정 |
| **MVP** | KPI 요약 위젯(AUM/마감건수/미납 캐피탈콜) + 긴급 Task 알림 + 빠른 이동 링크 |

---

## 2. 사용자 스토리 (User Stories)

| # | 제목 | 설명 | 우선순위 | 상태 |
|---|---|---|---|---|
| US-01 | KPI 위젯 조회 | 관리자는 전체 AUM, 운용 중인 조합 수, 이번 주 마감 Task 수를 한눈에 볼 수 있다. | 필수 | ✅ 완료 |
| US-02 | 긴급 Task 알림 | 관리자는 D-Day가 오늘이거나 지난 Task를 대시보드에서 즉시 확인하고 업무보드로 이동할 수 있다. (오늘 업무 카드에 지연 건수 병기) | 필수 | 🔄 변경됨 |
| US-03 | 캐피탈콜 미납 알림 | 관리자는 납입 기일이 경과했으나 미납된 LP 현황을 대시보드에서 볼 수 있다. | 필수 | 🟡 진행 중 |
| US-04 | 빠른 이동 | 관리자는 위젯/패널 CTA 클릭 한 번으로 업무보드/업무 파이프라인/펀드상세/LP관리 등 해당 페이지로 즉시 이동할 수 있다. | 필수 | 🔄 변경됨 |
| US-05 | 주간 캘린더 미리보기 | 관리자는 이번 주 예정된 이벤트(캐피탈콜, 총회 등)를 대시보드에서 미리볼 수 있다. | 선택 | ✅ 완료 |

---

## 3. 비기능 요구사항

- 첫 화면 로딩 시 모든 위젯 데이터를 **병렬(Parallel) API 호출**로 가져와 지연 없이 렌더링
- 대시보드 위젯은 **읽기 전용(Read-only)**, 수정 UI 일절 없음
