# PRD 05 — LP 관리 (LP Management)

> **상위 문서:** [PRD_MASTER.md](./PRD_MASTER.md)
> **관련 페이지:** `/lp-management`
> **상태:** 🟡 개발 중 | **최종 업데이트:** 2026-02-22

---

## 1. 기능 개요

| 항목 | 내용 |
|---|---|
| **목적** | 조합별 LP(유한책임조합원) 명부를 통합 관리하고, 시스템 전체에서 재활용하는 LP 주소록(Address Book)을 운영 |
| **가정** | 1. 하나의 LP 법인이 여러 조합에 걸쳐 출자할 수 있다. 2. LP 주소록은 조합 생성 시 자동완성 소스로 활용된다. |
| **범위 밖** | LP 포털(LP가 직접 로그인하는 외부 시스템), 디지털 서명 |
| **MVP** | 조합별 LP 명부 조회/편집 + LP 주소록 CRUD |

---

## 2. 사용자 스토리 (User Stories)

| # | 제목 | 설명 | 우선순위 |
|---|---|---|---|
| US-01 | LP 명부 조회 | 관리자는 조합별 LP 목록(이름/유형/약정액/납입액)을 조회할 수 있다. | 필수 |
| US-02 | LP 정보 편집 | 관리자는 LP의 이름, 유형, 연락처를 수정할 수 있다. | 필수 |
| US-03 | LP 약정액 수정 | 관리자는 LP의 출자 약정액을 수정할 수 있으며, 한글 금액 보조 텍스트가 실시간으로 표시된다. | 필수 |
| US-04 | LP 주소록 등록 | 관리자는 LP 법인의 기본정보(사업자번호/이메일/주소)를 주소록에 등록할 수 있다. | 필수 |
| US-05 | 주소록 자동완성 | 조합 생성 또는 LP 추가 시 주소록에서 LP를 검색하여 자동으로 정보를 채울 수 있다. | 필수 |
| US-06 | LP 삭제 | 관리자는 조합에서 LP를 삭제할 수 있다. (납입 이력이 있는 LP는 삭제 경고 표시) | 선택 |
