# V:ON 브랜드 가이드

## 서비스 정보
- **서비스명:** V:ON (브이온)
- **서브타이틀:** VC 조합 관리 플랫폼
- **한 줄 설명:** VC 관리역을 위한 올인원 펀드 운영 시스템

## 브랜드 네이밍 의미
- **V** = VC (Venture Capital)
- **:** = 디지털 시계의 콜론 (시간/기한 관리)
- **ON** = 켜다, 시작하다 (업무를 켜다)

## 컬러 팔레트

### Primary
| 이름 | HEX | 용도 |
|------|-----|------|
| Deep Navy | `#1E3A5F` | 로고, 텍스트, 헤더 배경 |
| Blue Accent | `#3B82F6` | 버튼, 링크, 강조 |
| Light Blue | `#60A5FA` | 호버, 보조 강조 |
| Sky | `#DBEAFE` | 배지, 배경 하이라이트 |

### Neutral
| 이름 | HEX | 용도 |
|------|-----|------|
| Dark Gray | `#1F2937` | 본문 텍스트 |
| Mid Gray | `#6B7280` | 보조 텍스트 |
| Light Gray | `#F3F4F6` | 카드/섹션 배경 |

|------|-----|------|

## 타이포그래피
- **한글:** Noto Sans KR (현재 적용됨)
- **영문/숫자:** Inter (로고 및 수치 표시)
- **로고 폰트:** Inter Bold 700 (V), Inter Medium 500 (ON)

## 로고 파일
- `/frontend/public/logo.svg` — 전체 워드마크 (V:ON)
- `/frontend/public/favicon.svg` — 아이콘형 (V + 도트)

## 로고 사용 규칙
- 최소 여백: 로고 높이의 50%
- 밝은 배경: Deep Navy 텍스트 + Blue 도트
- 어두운 배경: White 텍스트 + Light Blue 도트
