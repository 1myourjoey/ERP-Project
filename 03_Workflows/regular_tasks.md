## 4. 세금계산서 발행

**절차 (홈택스):**
1. 전자세금계산서 발행
2. 계산서 건별 발급 → 계산서(면세)
3. 공급받는 자 등록번호 및 상호 확인
4. 이메일 작성
5. 하단 작성일자 기입
6. 하단 품목명 기입 / 수량1 / 공급가액 입력
7. 영수증 청구

**인증서:** 세종 공인인증서 사용
