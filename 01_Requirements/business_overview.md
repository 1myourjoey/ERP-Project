# VC 백오피스 1인 관리 ERP — 업무 명세서

> 벤처캐피탈 백오피스 업무의 **누락 제로**를 목표로 한 통합 관리 시스템
> 참고: VCworks (guide.vcworks.kr) 기능 체계 + 실무 프로세스 기반
> 작성일: 2026-02-13

---

## 설계 원칙

1. **누락 방지 최우선** — 모든 업무는 워크플로우로 정의하고, 서류·기한·후속 작업을 자동 추적
2. **유기적 연결** — 조합 → 투자 → 서류 → 회계 → 보고가 하나의 흐름으로 연결
3. **1인 최적화** — 결재 라인 없이 실행 중심, 하나의 대시보드에서 모든 현황 파악
4. **이력 완전성** — 모든 변동은 기록으로 남기고, 교훈·패턴을 학습 자료로 축적

| 필드 | 설명 | 비고 |
|------|------|------|
| LP명 | 출자자명 | 필수 |
| LP 유형 | 개인 / 법인 / 모태펀드 / 연기금 / 기타 | 필수 |

**LP별 추적 항목:**
- 출자 이력 (출자 회차별 납입일, 금액)
- 배분 이력 (배분 회차별 원금, 수익, 성과보수)
- 출자증서 / 출자확인서 발급 이력
- LP 보고 발송 이력
