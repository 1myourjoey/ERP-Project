# 투자 체크리스트

> 투자 프로세스 전체 체크리스트
> 작성일: 2026-02-12

---

## 1단계: 투심위 체크리스트

### 투심위 전 (D-7 ~ D-1)

- [ ] 추가출자금 납입 통보 발송
- [ ] 납입 요청 공문 작성
- [ ] 조합 통장사본 첨부
- [ ] 투심 보고서 작성 완료
- [ ] 투자 계약서 초안 완료
- [ ] 준법감시보고서 작성 (3종 중 선택)
- [ ] 투심 전 ERP 등록 완료

### 투심위 당일 (D-day)

- [ ] 투심위 개최
- [ ] 투심위 결과보고서 작성
- [ ] 투심위 의사록 작성
- [ ] 서류 사인 받기 (13:00까지)
- [ ] 날인 확인 (결과보고서, 의사록)
- [ ] 투심 후 ERP 등록
- [ ] 기여율 정확히 입력
- [ ] 밸류 정확히 입력
- [ ] 농금원 ERP 결과보고서 등록

### 투심위 후 (D+1 ~ D+2)

- [ ] 투심위 회의록 작성 (2일 이내)

## 2단계: 투자계약 체결 체크리스트

### 계약 전 (D-3 ~ D-1)

- [ ] 권리관계서류 요청 (프론트 → 피투자사)
- [ ] 투자계약서 3부 준비
- [ ] 투자계약서 천공 완료
- [ ] 법인인감증명서 2부 준비 (피투자사, 이해관계자)
- [ ] 사용인감계 2부 준비
- [ ] 피투자사 법인계좌 확보

### 계약 당일 (D-day)

- [ ] 투자계약 체결
- [ ] 간인 완료 (순서: 조합 → 피투자사 → 이해관계인)
- [ ] 인감 날인 완료
- [ ] 신주식(전환사채)청약서 작성
- [ ] 인수증 작성
- [ ] 계약서 배부 (가운데 양옆: 피투자사, 맨 오른쪽: 이해관계인)
- [ ] 계약서 보관/등록
- [ ] 투자계약서 날인본 ERP 업로드
  - [ ] LP보고 → 투자 → 투자계약서 업로드
  - [ ] 투자관리 → 계약정보 등록/재원확정(승인)
  - [ ] 투자집행 → 투자금 인출요청(원장생성)
- [ ] 투자금 출금 확인

### 계약 후 (D+1 ~ D+2)

- [ ] 운용지시서 작성
- [ ] 운용지시서 서류 확인
  - [ ] 운용지시서
  - [ ] 사업자등록증
  - [ ] 계약서 제1조 발췌
  - [ ] 의무기재사항 확인서 (준법감시인 날인)
  - [ ] 투심위 의사록/결과보고서 (날인본)
  - [ ] 주주명부 (투자 전)
- [ ] 은행명 정확히 기입 확인
- [ ] 운용지시 실행 (계약 후 2일 이내)

## 3단계: 투자 후 서류처리 체크리스트

### 바이블 제작 (D+3 ~ D+7)

- [ ] 투자 전 서류 취합
- [ ] 권리관계서류 확인 (프론트 전달)
- [ ] 투자 후 서류 취합
  - [ ] 주주명부 (변경 후)
  - [ ] 등기부등본 (증자 후, 말소사항 포함)
  - [ ] 주권미발행확인서
  - [ ] 주금납입영수증
  - [ ] 주식납입금 보관증명서
- [ ] 투자 당일 서류 취합 리스트 확인
- [ ] 서류모음 재확인 및 바이블화
- [ ] 고객거래확인서 구분 확인
- [ ] 주민번호 뒷자리 블러처리

### 수탁사 실물 송부 (D+15)

- [ ] 등기부등본 확인
- [ ] 주식 수와 계약서 일치 여부 검증
- [ ] 주권미발행확인서 준비
- [ ] 법인인감증명서 준비 (3개월 이내)
- [ ] 등기부등본 준비 (변경 후)
- [ ] 주주명부 준비 (변경 후)
- [ ] 실물 송부 완료 (투자일로부터 15일 이내)

### 기타 후속 처리 (D+10 ~ D+21)

- [ ] VICS 기업등록 (프로젝트 시)
- [ ] 등록원부 변경 신청 (LP 회람)
  - [ ] 변경등록신청서
  - [ ] 조합원 명부 (날인본)
  - [ ] 출자금 납입 확인서
  - [ ] 납입 요청 공문
- [ ] 출자증서 발급 (변경 등록 후)
