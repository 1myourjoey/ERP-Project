## 3. 운용지시서 작성

**배경:** 2026-02-11 운용지시 실행

### 필수 확인 사항
- ⚠️ **은행명 정확히 기입** (실수 시 재작성 필요)
- 피투자사 법인계좌 사전 확보 권장

### 필수 서류
1. 운용지시서
2. 사업자등록증
3. 계약서 제1조 발췌
4. 의무기재사항 확인서 (준법감시인 날인)
5. 투심위 의사록/결과보고서 (날인본)
6. 주주명부 (투자 전)

## 4. 투자 후 서류 관리

**배경:** 2025-11-21, 2026-01-08 경험

### 핵심 원칙
- 규약상 **영업일 15일 이내** 수탁사 송부 원칙
- 피투자사 사정 고려한 유연한 대응 가능 (패널티 없음)
- 빠른 처리 필요

### 날인 필수 서류
1. 투심위 결과보고서
2. 투심위 의사록
3. 준법감시보고서

### 실무 팁
- 원칙: ERP 업로드부터 준법감시보고서는 날인본
- 실무: 편의상 투자 후 의사록과 함께 날인본 업로드 가능

## 10. 세금계산서 발행 절차

**배경:** 2025-12-30 세금계산서 발행

### 홈택스 발행 절차
1. 전자세금계산서 발행
2. 계산서 건별 발급 → 계산서(면세)
3. 공급받는 자 등록번호 및 상호 확인
4. 이메일 작성
5. 하단 작성일자 기입
6. 하단 품목명 기입 / 수량1 / 공급가액 입력
7. 영수증 청구

**인증서:** 세종 공인인증서 사용

## 11. 투자계약서 ERP 업로드

**배경:** 2026-02-10 ERP 업로드 시간 초과

### 교훈
- ERP 등록은 단순 업로드가 아닌 **3단계 프로세스**
- 예상 시간 10m → 실제 45m (약 4.5배)

### 3단계 프로세스
1. LP보고 → 투자 → 투자계약서 업로드
2. 투자관리 → 계약정보 등록/재원확정(승인)
3. 투자집행 → 투자금 인출요청(원장생성)
